Icons

One curated set on a 24×24 grid, single 2px stroke, currentColor. Server-rendered templates use the $$:icon() macro; Lit components use the <ol-icon> element. Both draw from the same build-time sprite, so a glyph looks identical either way.

Usage

$$:icon("search")
$$:icon("globe", size="lg")
$$:icon("globe", label=_("Language"))

<ol-icon name="search"></ol-icon>
<ol-icon name="globe" size="lg" label="Language"></ol-icon>
