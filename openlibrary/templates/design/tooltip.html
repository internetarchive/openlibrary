Tooltip component

The ol-tooltip component displays contextual information on hover or focus. It supports configurable placement, show delay, and arrow visibility. The tooltip appears after a short delay and is shown and hidden instantly.

Basic tooltip

Hover or focus the button to see the tooltip. Default placement is top with a 150ms show delay.

<ol-tooltip content="Edit this item">
  <button>Edit</button>
</ol-tooltip>

Placement options

Use placement to control where the tooltip appears. It automatically flips when near viewport edges. Add arrow to show the directional arrow (hidden by default).

<ol-tooltip content="I'm on top" placement="top" arrow>
  <button>top</button>
</ol-tooltip>

Rich content via slot

Use slot="content" for HTML content inside the tooltip.

<ol-tooltip>
  <button>Info</button>
  <span slot="content">Rich <strong>HTML</strong></span>
</ol-tooltip>
Keyboard shortcut: Ctrl+K

Book hovercard

Slotted content lives in the light DOM, so it is styled by page CSS, not the component. Inherited properties like color cascade in from the tooltip; structural styles (weight, size, muted text) come from the consuming page's stylesheet.

<ol-tooltip placement="bottom" arrow>
  <img class="book-tip__trigger" src="…">
  <div slot="content" class="book-tip">…</div>
</ol-tooltip>
