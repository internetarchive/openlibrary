Carousel

The ol-carousel component is a paged, Netflix-style horizontal carousel. Items are passed as direct children; the component sets their width from responsive breakpoints and provides arrow buttons, edge peek, and bar-segment page indicators. All motion uses spring physics.

Basic usage

Pass each item as a direct child. Set an accessible label for the carousel region.

<ol-carousel label="Trending">
  <div>...item...</div>
  <div>...item...</div>
</ol-carousel>

Hidden indicators

Add hide-indicators to drop the page indicator bar (e.g. when the surrounding context already signals position).

Page-change event

The carousel fires an ol-carousel-page-change event after each page transition, with { page, totalPages } in event.detail.

<ol-carousel id="demo-carousel" label="Popular">
  ...items...
</ol-carousel>

<script>
  carousel.addEventListener('ol-carousel-page-change', (e) => {
    console.log(e.detail.page, e.detail.totalPages);
  });
</script>
