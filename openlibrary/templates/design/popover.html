Popover component

The ol-popover component provides a reusable animated popover panel anchored to a trigger element. It animates from the trigger using transform-origin, closes on Escape or outside click, and respects prefers-reduced-motion.

Menu popover

Popovers animate from the trigger using transform-origin. Click the button to see the scale + fade entrance.

<ol-popover>
  <ol-button slot="trigger">Options</ol-button>
  <ul>...</ul>
</ol-popover>
