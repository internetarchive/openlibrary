Select Popover

The ol-select-popover component pairs a trigger button with a popover containing a multi-select list and an optional filter input. Selected items group above unselected ones; group order stays stable while open and re-sorts when reopened.

Default usage with auto-search (12 items)

With more than 8 items, a filter input renders automatically. Selected items move to a SELECTED group on next open.

<ol-select-popover
  label="Language"
  placeholder="Filter languages…"
  unselected-heading="LANGUAGES"
  items='[{"value":"en","label":"English"}, ...]'>
</ol-select-popover>

Selected: (none)

Small list, no search input

With 8 items or fewer, the filter input is hidden by default. Set search-threshold to a different number to override (use 0 to always show, a large value to always hide).

<ol-select-popover
  label="Layout"
  unselected-heading="LAYOUTS"
  items='[{"value":"grid","label":"Grid"}, ...]'>
</ol-select-popover>

Custom trigger via slot

Provide your own trigger element with slot="trigger". The component handles open/close and ARIA wiring.

<ol-select-popover label="Genre" items="...">
  <a slot="trigger" href="#">Pick a genre</a>
</ol-select-popover>
Pick a genre

With ol-chip-group showing selections

Selected items render as chips in an ol-chip-group next to the trigger. Clicking a chip's close icon removes that selection from the popover.

<div class="row">
  <ol-select-popover label="Language" ...></ol-select-popover>
  <ol-chip-group>
    <ol-chip selected>English</ol-chip>
  </ol-chip-group>
</div>
