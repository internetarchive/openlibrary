Every piece of Open Library is defined by the type of content it displays, usually by content definition (e.g. Authors, Editions, Works) but sometimes by content use (e.g. macro, template, rawtext). Changing this for an existing page will alter its presentation and the data fields available for editing its content. Please use caution changing Page Type!

(Simplest solution: If you aren't sure whether this should be changed, don't change it.)
