# Open Library API Documentation

Welcome to the official developer documentation for the Open Library APIs.

Open Library provides a wide range of public APIs to help developers build applications using our catalog of millions of books, authors, and subjects.

## Get Started

- **[Getting Started](./api/getting-started.md)**: Make your first API call in minutes.
- **[Authentication](./api/authentication.md)**: Learn about the mandatory `User-Agent` header requirement.
- **[API Reference](./api/search.md)**: Explore endpoints for search, books, authors, and more.
- **[Error Handling](./api/errors.md)**: Understand HTTP codes and rate limiting.

## Key APIs

- **[Search API](./api/search.md)**: Search for books by title, author, or keyword.
- **[Books API](./api/books.md)**: Get detailed information about specific works or editions.
- **[Authors API](./api/authors.md)**: Fetch author biographies and lists of their works.
- **[Covers API](./api/covers.md)**: Access millions of book cover images in multiple sizes.
